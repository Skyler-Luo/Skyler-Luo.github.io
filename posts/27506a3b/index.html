项目部署

克隆项目并搭建环境

先更新Ubuntu系统上的软件包

接着安装Python及相关工具

使用 SSH 连接 GitHub，将自己的项目克隆到服务器上

  • 生成 SSH 密钥

按照提示生成 SSH 密钥，并保存到默认位置（通常是 ~/.ssh/id_rsa）。

  • 添加 SSH 密钥到 GitHub

打开生成的公钥文件，并复制内容

登录到 GitHub，前往 Settings -> SSH and GPG keys -> New SSH key，粘贴公钥并保存。

  • 将项目克隆到本地

创建并进入你想存放项目的目录，然后从 GitHub 中拉取项目代码

  • 创建虚拟环境并安装库

进入到项目的根目录后，创建环境并激活
